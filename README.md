# Deps
* python3x
* requests (pip install requests)
* beautifulsoup (pip install beautiful soup)
* geojson (pip install geojson)
* flask (pip install flask)

# To run

### Flask
* cd to api directory
* python api.py

### Website
* cd to front-end
* npm run start

### Generate CSV
* cd to scraper
* python scrape.py

# Output
* output.csv (in scraper dir)
